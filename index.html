# Welcome!

The release of AWS App Mesh on March 27, 2019, provides developers with application-level networking for their cloud applications running on AWS. App Mesh helps developers run and monitor their HTTP and TCP services, which may be running on different compute services, at scale.

My colleagues, Mridula Ghrandhi, Michael Hausenblas, and I have written this guide to help developers get the most out of App Mesh. The guide is open source, so feel free to comment (open issues) or submit pull requests to the repo. You can also reach out to me on Twitter. My account is open to direct messages.
